### 查找array 中某个元素的位置

```Python
index = np.where(A==a)[0]
```

### 查找list 中某个元素的位置


```Python
A.index(a)
```

注意： array的np.where只能返回某一个元素的数值； list可以实现一组点列中删选出第一个出现该点（三维坐标）的位置值



### 将list中的元素转为str类型
方法一：

```Python
>>> L = [1,2,3,4,5]
>>> map(str, L)
['1', '2', '3', '4', '5']
```

```Python
a = [1,2,3]
b = [ str(i) for i in a ]
```
